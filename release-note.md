## 0.6.2

### Minor Changes

- **新增返回顶部与随机文档**：两个新的内置动作，一键回到文档顶部，或随机打开一篇笔记漫游。
- **AI 脚本上下文全量公开**：新增 `get-script-context` MCP 工具，AI 助手现在可以查询 `plugin`、`siyuan` SDK、`window.siyuan` 的完整 API 参考，以及常见错误写法对照，写脚本时不再靠猜。
- **动作类型选择更智能**：`get-action-schema` 现在动态列出所有实际注册的内置命令，以及思源系统命令和第三方插件命令，AI 生成动作时自动按优先级（内置 > 系统命令 > 第三方命令）选择最合适的类型。
