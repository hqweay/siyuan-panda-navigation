## 0.5.0

### Minor Changes

- **写脚本如有神助**：在编写自定义脚本时，现在可以直接使用强大的 `utils` 对象啦！它打包了插件自带的所有“神技”（比如快速查询笔记、打开专属面板等）。以后无论是你自己写，还是让 AI 帮你写复杂脚本，都极其方便，再也不用从零查底层 API 了！
- **AI 智能体全自动“读心术”**：我们在底层加入了超酷的自动化代码提取机制。现在开发者只要加上一行新功能，系统就会全自动把这些工具暴露给 AI（通过 MCP 协议）。这意味着，未来你让 AI 帮你定制熊猫导航按钮时，AI 瞬间就能知道咱们插件有哪些现成的魔法可以直接用，懂你所需！
