## 0.4.2

### Minor Changes

- **执行自定义 JS 脚本**：新的"脚本"动作类型让熊猫导航支持在思源环境中直接运行原生 JavaScript！你可以使用 `plugin` 和 `siyuan` 变量调用思源内核 API、操作 DOM、发送网络请求等，支持顶层 `await` 和完整的异步操作。
- **智能参数输入控件**：所有内置命令现在支持根据实际需要自动切换输入控件——有的用单行文本框（如 URL），有的用多行文本域（如 SQL、脚本），有的用下拉选择框（如数据库）。设置体验更顺手，不再一刀切。

### Patch Changes

- **SQL 随机漫游重命名**：为体现 SQL 随机漫游的高阶玩法，重命名为"高级随机漫游 (SQL)"，并改用大文本框输入，写复杂查询更舒心。
- **新增预设模版**：设置菜单新增"自定义脚本示例"预设，展示如何用脚本引擎实现弹窗提示、查询数据库并打开文档等一系列流程，开箱即用。
