# 🐼 熊猫导航

专为思源笔记打造的移动端/桌面端自适应导航栏，让你在思源中浏览文档时操作更顺手。

![桌面端效果](./screenshots/pc.png)
![移动端效果](./screenshots/mobile.png)

## 功能亮点

- **移动端药丸导航**：底部悬浮导航栏，向下滚动时自动缩小为小药丸，不遮挡内容；点击即可展开
- **桌面端简洁导航**：纯图标导航栏，悬停显示名称提示，干净不占空间
- **预设配置库**：内置多种精美模板（如九宫格基础导航、常用动作演示等），一键无脑导入，新手极其友好
- **完全自由的数据结构**：不再受限于死板的两级菜单，通过“分组（Group）”和“动作（Action）”的组合，你可以拼搭出任何想要的布局
- **标签文字开关**：可以控制按钮下方是否显示文字标签，移动端/桌面端分开设置
- **多端设备独立控制**：每个按钮都可以独立设置仅在移动端、仅在 PC 端或双端显示

## 快速开始

在思源笔记集市中搜索「熊猫导航」安装即可。第一次安装时，插件会自动为您应用一套精美的 **默认配置**，包含日常高频的导航功能！

## 配置说明

### 基础设置
- **启用悬浮导航栏**：选择在哪些设备上显示导航栏
- **按钮标签文字**：控制按钮下方是否显示文字说明

### 高级与维护
- **导入预设**：直接选择“基础导航网格”等预设配置，一键生成。
- **恢复默认配置**：如果在自定义菜单的过程中弄乱了布局，可以在导入预设下拉框中选择此项，瞬间重置回出厂设定。

### 快捷动作与分组

你可以自由创建“动作”，或把多个动作塞入“分组”中（甚至指定该分组以网格模式展示），支持以下操作类型：

| 类型 | 说明 |
|------|------|
| 链接 / 文档ID | 打开网页、思源内置链接或任意文档 |
| 随机 SQL | 按自定义规则随机跳转文档 |
| 内置命令 | 调用思源自带功能，如前/后退、上下级跳转、打开设置等 |
| 添加到数据库 | 把当前文档快速添加到指定的属性视图（数据库）中 |
| 调用系统命令 | 触发思源本体以及第三方插件的任意注册命令 (包括打开日记等) |

## 使用提示

- 移动端长按无标签的按钮会显示名称提示
- 导航栏上的「快捷操作」按钮可以展开子菜单，里面放不常点但偶尔要用到的动作
- 桌面端导航栏始终显示，不会跟随滚动隐藏
