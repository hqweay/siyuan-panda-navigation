# 🐼 熊猫导航

专为思源笔记打造的移动端/桌面端自适应导航栏，让你在思源中浏览文档时操作更顺手。

## 功能亮点

- **移动端药丸导航**：底部悬浮导航栏，向下滚动时自动缩小为小药丸，不遮挡内容；点击即可展开
- **桌面端简洁导航**：纯图标导航栏，悬停显示名称提示，干净不占空间
- **快捷动作**：把常用操作放在导航栏上——打开首页、随机漫步、全局搜索、打开某个数据库…… 你说了算
- **按钮自由配置**：哪些按钮显示、显示在哪个设备、显示顺序，全由你决定
- **标签文字开关**：可以控制按钮下方是否显示文字标签，移动端/桌面端分开设置

## 快速开始

在思源笔记集市中搜索「熊猫导航」安装即可。

## 配置说明

### 基础设置![alt text](image.png)
- **启用悬浮导航栏**：选择在哪些设备上显示导航栏

### 按钮显示
- 每个按钮可以独立设置显示在「移动端」「桌面端」「两端都显示」或「不显示」
- 拖拽排序：调整按钮在导航栏上的排列顺序
- **按钮标签文字**：控制按钮下方是否显示文字说明

### 快捷动作

快捷动作是可以添加到导航栏或子菜单中的自定义操作，支持以下类型：

| 类型 | 说明 |
|------|------|
| 链接 / 文档ID | 打开网页、思源内置链接或任意文档 |
| 随机 SQL | 按自定义规则随机跳转文档 |
| 内置命令 | 调用思源自带功能，如全局搜索、切换文件树、打开设置等 |
| 添加到数据库 | 把当前文档快速添加到指定的属性视图（数据库）中 |
| 打开插件设置 | 快速打开熊猫导航的设置面板 |

每个动作可以设置在「导航栏」上直接显示，或放在「子菜单」中。

## 使用提示

- 移动端长按无标签的按钮会显示名称提示
- 导航栏上的「快捷操作」按钮可以展开子菜单，里面放不常点但偶尔要用到的动作
- 桌面端导航栏始终显示，不会跟随滚动隐藏
