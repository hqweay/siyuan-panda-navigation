# 🐼 熊猫导航

专为思源笔记打造的移动端/桌面端自适应导航栏，让你在思源中浏览文档时操作更顺手。

仓库地址：[GitHub - hqweay/siyuan-panda-navigation: 思源插件：熊猫导航 · GitHub](https://github.com/hqweay/siyuan-panda-navigation)

## 功能亮点

- **桌面端随心拖拽**：纯图标悬浮导航，不仅不占空间，更支持**全局随意拖拽**。你可以把它放在屏幕的任何角落，底层采用极致的 120FPS 原生 DOM 硬件加速，顺滑跟手零延迟，并自动记忆位置。
- **移动端药丸导航**：底部悬浮导航栏，向下滚动时自动变身为贴边小药丸，绝不遮挡你宝贵的笔记内容；轻轻一点即可展开。
- **灵动岛式交互体验**：全面重构的底层动画引擎。在展开菜单和切换分组时，导航面板将如“灵动岛”一般丝滑平移、自适应伸缩，现代感拉满。
- **像搭积木一样的自由菜单**：彻底打破死板的层级限制，你可以自由创建分组（还能设为网格模式），把喜欢的操作塞进去，想怎么排就怎么排。
- **一键导入配置模版**：觉得从头配置太麻烦？插件自带多款精美"预设模版"（比如超级好用的九宫格基础导航），一键导入，新手也能瞬间上手。
- **多端设备独立控制**：你可以让某个按钮"仅在手机端显示"，或者"仅在电脑端显示"，还能单独控制各端的标签文字显示，完美适配不同设备的阅读习惯。

## 轻松配置

### 基础外观设置
- **启用悬浮导航栏**：自由选择是在手机端、电脑端还是两端同时显示导航栏。
- **按钮标签文字**：随时控制按钮下方是否显示文字说明。

### 高级玩法与重置
- **导入预设**：不知道怎么布局好看？直接在设置里选择"基础导航网格"等预设，一键拥有完美排版。
- **恢复默认配置**：如果平时折腾菜单不小心弄乱了，可以在预设下拉框里找到这个选项，点一下就能瞬间恢复出厂设定，不用有任何心理负担。

### 快捷动作与分组组合

你可以把常用操作做成"动作"，或者把它们丢进"分组"里折叠起来。插件支持的超强操作类型：

| 类型 | 说明 |
|------|------|
| **内置功能 (Builtins)** | 插件自带的超级加强版功能，包括：<br/>• **自定义链接/文档ID**：快速打开外部网页或思源某篇文档，完美兼容移动端<br/>• **随机漫游 (SQL)**：用自定义 SQL 从浩瀚的笔记海里随机翻阅<br/>• **文档层级漫游**：快速跳转父级、子级、相邻文档等<br/>• **添加到数据库**：一键把当前文章塞进你指定的属性视图里<br/>• **执行自定义 JS 脚本**：在思源环境中直接运行原生 JavaScript，支持顶层 await 和异步操作，可调用 `plugin` 和 `siyuan` 等变量（比如用 `siyuan.fetchSyncPost` 调用内核 API 或手动打开指定文档） |
| **系统命令 (System)** | 直接触发思源笔记本体自带的任意系统快捷命令 |
| **第三方命令 (Plugin)** | 神仙功能！能直接触发你安装的**其他思源插件**里注册的任意命令（例如一键打开"今日日记"等） |

### 🤖 拥抱未来：原生的 MCP AI 智能体支持

你想通过“执行自定义 JS 脚本”实现一些酷炫的自动化操作，但却**完全不懂编程**？

没关系！熊猫导航在底层实现了革命性的 **MCP (Model Context Protocol)** 协议打通。这不仅是一项技术突破，更是送给每一位普通用户的高阶福利。当你在使用支持 MCP 的主流 AI 助手时：

1. **AI 自动感知能力边界**：AI 会自动“阅读”熊猫导航的底层架构，精准识别出插件为你准备的所有快捷工具（如数据库查询、弹窗渲染、文档树遍历等）。
2. **“动口不动手”的极客体验**：你只需要对 AI 说：“帮我写一个一键统计今天所有待办事项数量并弹窗的熊猫导航按钮”，AI 就能利用它获取到的专属 API 字典，直接为你写出百分百兼容的 JS 脚本，甚至指导你一键配置。

这使得熊猫导航不仅是一个 UI 工具，更是一个向 AI 开放的无限潜能基座。零代码基础的你，也能随时定制最硬核的效率流程！

## 🛠 面向开发者：插件底层机制

熊猫导航的核心命令模块设计遵循**高内聚、低耦合**的注册表模式。无论是想要新增内置功能，还是为 AI 提供新工具，整个流程都已被高度自动化。

### 1. 扩展内置命令
如果你需要向熊猫导航中注册新的系统级“内置动作”，步骤如下：

1. 进入 `src/builtins/commands/` 目录。
2. 新建 `.ts` 文件并实现 `BuiltinCommand` 接口并导出。

**动态 UI 控件绑定**：你可以通过 `inputType` 字段直接定义其在设置界面的 UI 交互形态。支持 `"text"`、`"textarea"` 以及 `"select"`。

**构建期自动注入**：底层的 AST 提取脚本结合 Vite 的动态扫描引擎，会在编译阶段全自动扫描命令并构建元数据。开发者只需专注业务逻辑，免除手动维护中央路由的困扰。

### 2. MCP 能力的自动暴露

对于 AI 开发者：插件不仅注入了暴露大量思源内核 API 的 `utils` 库，更原生打通了向大语言模型暴露能力的数据链路。
当你在 `commands` 或 `panda-utils.ts` 下增加新功能时，AST 引擎会自动提取其元数据，并通过 `panda-nav:get-capabilities` 工具注册至 MCP。AI 会自动感知到你新加的工具链，实现**零配置的智能 API 联想与代码生成**。
