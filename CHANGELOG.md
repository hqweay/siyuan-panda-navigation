# siyuan-panda-navigation

## 0.6.3

### Minor Changes

- **化身移动端格式刷**：现在可以把思源笔记自带的“编辑器命令”（例如加粗、斜体、各级标题、列表等）直接添加到导航栏了！这意味着你可以把它变成一个随时悬浮在手边的专属排版工具栏，尤其在手机和平板上打字排版时非常方便。
- **设置界面分类优化**：动作配置界面里，系统命令和编辑器命令现在都支持了清晰的分组展示（比如“插入”、“表格”、“排版”等），找起功能来再也不用眼花缭乱。
- **一键切换预设 (Workflow Switcher)**：新增了内置命令“切换导航预设”。你可以把它配在动作菜单里，填入某个自定义预设或系统内置预设的名称，点击后瞬间在多个界面布局间无缝切换（例如“阅读模式”、“写作模式”），配合分组功能使用效果更佳！

### Patch Changes

- 修复了配置过长导致的列表显示异常问题。
- 修复了导航按钮点击时导致编辑器失去焦点的问题。

## 0.6.2

### Minor Changes

- **新增返回顶部与随机文档**：两个新的内置动作，一键回到文档顶部，或随机打开一篇笔记漫游。
- **AI 脚本上下文全量公开**：新增 `get-script-context` MCP 工具，AI 助手现在可以查询 `plugin`、`siyuan` SDK、`window.siyuan` 的完整 API 参考，以及常见错误写法对照，写脚本时不再靠猜。
- **动作类型选择更智能**：`get-action-schema` 现在动态列出所有实际注册的内置命令，以及思源系统命令和第三方插件命令，AI 生成动作时自动按优先级（内置 > 系统命令 > 第三方命令）选择最合适的类型。

## 0.6.1

### Minor Changes

- **新增自定义预设功能**：在设置面板新增"另存为预设"按钮，可以一键保存当前全部的导航按钮和分组。
- **灵活的预设加载策略**：加载自定义预设时新增原生弹窗确认，你可以自由选择"覆盖"当前菜单，还是将预设"追加"为新的独立分组。
- **扩展 AI 布局能力**：底层新增面向 AI 的预设管理接口。AI 客户端现在可以直接帮你备份预设、并随时切换不同的专属导航布局。

## 0.6.0

### Major Changes

- **桌面端支持自由拖拽**：桌面端导航栏现在支持随意拖拽，拖动流畅无延迟。松开后会自动记住位置，并且在调整窗口大小时会自动吸附，避免导航栏跑到屏幕外。
- **面板切换动画升级**：优化了多分组切换时的子菜单动画，现在面板会平滑移动并自动调整宽度。

### Patch Changes

- 修复：展开状态下点击其他分组按钮时，面板可能会意外关闭的问题。
- 修复：iOS 等移动端环境下，内置的 URL 命令无法正常跳转打开网页的问题。
- 修复：拖拽导航栏时如果被其他操作打断（如弹窗），可能导致导航栏一直卡在拖拽状态的问题。

## 0.5.0

### Minor Changes

- **新增脚本工具包**：在执行“自定义脚本”动作时，可以直接使用自带的 `utils` 工具库，大幅降低编写复杂功能的门槛。
- **拥抱 AI 时代**：插件的底层能力现已自动向外部 AI 开放。使用支持该协议的 AI 客户端时，你可以直接让 AI 帮你编写定制化的复杂脚本。

## 0.4.2

### Minor Changes

- **执行自定义 JS 脚本**：新的"脚本"动作类型让熊猫导航支持在思源环境中直接运行原生 JavaScript！你可以使用 `plugin` 和 `siyuan` 变量调用思源内核 API、操作 DOM、发送网络请求等，支持顶层 `await` 和完整的异步操作。
- **智能参数输入控件**：所有内置命令现在支持根据实际需要自动切换输入控件——有的用单行文本框（如 URL），有的用多行文本域（如 SQL、脚本），有的用下拉选择框（如数据库）。设置体验更顺手，不再一刀切。

### Patch Changes

- **SQL 随机漫游重命名**：为体现 SQL 随机漫游的高阶玩法，重命名为"高级随机漫游 (SQL)"，并改用大文本框输入，写复杂查询更舒心。
- **新增预设模版**：设置菜单新增"自定义脚本示例"预设，展示如何用脚本引擎实现弹窗提示、查询数据库并打开文档等一系列流程，开箱即用。

## 0.4.1

### Patch Changes

- **修复问题**：解决了上个版本中由于文档树接口问题，导致“跳转父级”、“跳转子文档”、“上一篇/下一篇”这四个快捷命令失效的 Bug。现在它们又满血复活啦！

## 0.4.0

### Minor Changes

- **真正自由的自定义导航**：彻底打破了旧版本死板的菜单限制，现在的导航菜单就像“搭积木”一样自由！你可以随意创建多个导航分组，把各种快捷操作放进去，甚至支持多级菜单的无限嵌套。
- **全新预设模版，一键导入**：觉得配置太麻烦？现在插件内置了精美且高度实用的“基础导航网格”等预设模板。哪怕你是刚安装的新手，只需点一下“导入预设”，瞬间就能获得完美配置！
- **一键恢复默认配置**：如果平时不小心把导航菜单弄乱了，不用担心。在预设菜单的最底下有一个红色的“恢复默认配置”选项，点一下就能变回出厂设置。
- **界面体验大升级**：
  - 完美对齐了移动端弹出菜单和底部药丸导航的高度，视觉效果更和谐。
  - 修复了在电脑端弹窗里，图标和文字混在一起时出现的底部高低不平的问题，现在看起来非常清爽。
  - 移动端药丸菜单重新加入了灵动的物理动画，点开和收起的过渡变得极其丝滑。

## 0.3.0

### Minor Changes

- 二级菜单图标面板：导航按钮和快捷动作的二级菜单现支持「图标面板」展示模式（设置 → 子菜单展示方式），图标居中排列，桌面端不会因按钮少而挤压换行

- 视觉优化：导航栏圆角统一为 20px 药丸形状，展开时与二级栏无缝衔接；按钮过渡动画优化
- 前进 / 后退按钮默认仅在移动端显示
- 移动端滚动监听兼容性优化
