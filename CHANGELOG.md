# siyuan-panda-navigation

## 0.4.0

### Minor Changes

- **配置模型大重构**：打破过去死板的两级结构限制。现在“导航菜单”和“今日日记”不再作为特殊黑盒，而是彻底转换为自由组合的“分组（Group）”和“动作（Action）”。你现在可以像搭积木一样随心所欲地组织导航层级。
- **预设配置库（Presets）上线**：新增“导入预设”功能！内置精美的“基础导航网格”和“快捷动作”组合。不仅对于新手开箱即用，老用户也能一键导入参考。
- **一键恢复出厂设置**：如果配置弄乱了，可以在“导入预设”菜单的最底端找到红色的“恢复默认配置”选项，确认后即可瞬间满血复活（回到干净好用的网格状态）。
- **统一系统命令调用**：废弃了之前专门为“今日日记”等写的独立封装逻辑，现在统归于底层强大的系统命令引擎。大大降低了历史包袱，执行更高效。
- **UI 像素级调优**：
  - 精准对齐了移动端“抽屉弹出层”与主导航栏的高度（精准 56px）。
  - 精确压实了 PC 端弹窗悬浮网格的高度，去除了 SVG 和 Emoji 混排时的底部留白差异。
  - 移动端药丸菜单重新加入 `svelte/transition` `fly` 动画引擎，出入更加顺滑物理。
  - 彻底消灭了设置界面的可访问性（A11y）编译警告。

## 0.3.0

### Minor Changes

- 二级菜单图标面板：导航按钮和快捷动作的二级菜单现支持「图标面板」展示模式（设置 → 子菜单展示方式），图标居中排列，桌面端不会因按钮少而挤压换行
- 快捷动作优化：无自定义动作时自动隐藏快捷操作按钮；每个动作可独立配置显示设备和位置（导航栏 / 子菜单）
- 随机漫游性能优化：限制 SQL 查询返回行数，减少全表扫描量
- 视觉优化：导航栏圆角统一为 20px 药丸形状，展开时与二级栏无缝衔接；按钮过渡动画优化
- 前进 / 后退按钮默认仅在移动端显示
- 移动端滚动监听兼容性优化

## 0.2.0
